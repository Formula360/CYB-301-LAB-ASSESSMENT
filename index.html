Cryptography
Lab Assignment

This assignment covers the implementation of RSA asymmetric encryption/decryption and AES symmetric encryption using Python.

Task 01

Implementing RSA Encryption & Decryption

Implement a simple RSA encryption and decryption system using Python.

Instructions

Write a Python program that:

  • Generates two random prime numbers (p and q).
  • Computes the public key (n, e) and private key (n, d).
  • Encrypts a user-inputted message using the public key.
  • Decrypts the encrypted message using the private key.
  • Uses probabilistic primality testing (e.g., Fermat's or Miller-Rabin test) to generate large prime numbers.

Expected Output

Terminal Output

Generated Keys: p = 104729 | q = 224737 n = 23535043673 e = 65537 (public exponent) d = 18473920513 (private exponent) Encryption: Plaintext : "Hello, World!" Ciphertext : 14829301... [integer representation] Decryption: Decrypted : "Hello, World!" ✓
Task 02

Implementing AES Symmetric Encryption

Objective: Implement AES encryption and decryption using Python's cryptography library.

Instructions

Install the required library, then write a Python program that:

  • Accepts a plaintext message from the user.
  • Generates a random AES key.
  • Encrypts the message using AES-128 in CBC mode.
  • Decrypts the ciphertext to retrieve the original message.

Expected Output

Terminal Output

AES-128 CBC Mode: AES Key (hex) : 3a4f2b1c8d9e0f5a6b7c8d9e0f1a2b3c IV (hex) : 1a2b3c4d5e6f708192a3b4c5d6e7f809 Encryption: Original Message : "Secret Data 2024" Ciphertext (hex) : 7f8a9b0c1d2e3f4a5b6c7d8e9f0a1b2c... Decryption: Decrypted Message : "Secret Data 2024" ✓
